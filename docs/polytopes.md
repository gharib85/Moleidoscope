Polytopes
=========

In elementary geometry, a polytope is a geometric object with "flat" sides.
Here, only polyhedra and polygon are considered.


Library
-------

Current library includes following polytopes:
-   Cube
-   Tetrahedron
-   Octahedron
-   Triangle
